Presentación servidor web

¿Qué es un servidor web?

Un servidor web o servidor HTTP es un programa informático que procesa una aplicación del lado del servidor, realizando conexiones con el cliente y generando o cediendo una respuesta en cualquier lenguaje o Aplicación del lado del cliente. El código recibido por el cliente es renderizado por un navegador web.
Instalación servidor web en Linux
Comandos a usar:
1.- apt-get update
2.- apt-get upgrade
3.- apt-cache policy apache2
4.- apt-get install apache2
Personalizacion de la página inicial:
1.- Editar el archivo /var/www/html/index.html y poner lo que desee.
Abrir puertos en el router:

1.- Hemos abierto nuestro router Movistar en el navagador (192.168.1.1)
2.- Hemos configurados el puerto 80 tcp,udp y le hemos asignado la direccion
ip de nuestro servidor (192.168.1.41)
Darse de alta en servidor DNS

Hemos accedido a la pagina de no-ip y hemos registrado una cuenta para conseguir nuestra dirección DNS.
